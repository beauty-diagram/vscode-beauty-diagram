# Changelog

## 0.1.0 — first release

- Markdown Preview rendering for `mermaid` and `plantuml` fences via Beauty Diagram API
- 9 themes (Classic / Modern / Slate / Atlas / Obsidian / Brutalist / Atelier / Blueprint / Memphis)
- Per-block directive support (`bd:theme`, `bd:bg`)
- CodeLens: `↗ Open in Beauty Diagram editor` above every fence
- Source injection commands (inject current file / inject workspace / clean orphans)
- Verify API key command
- Self-host via `beautyDiagram.apiBase` setting
